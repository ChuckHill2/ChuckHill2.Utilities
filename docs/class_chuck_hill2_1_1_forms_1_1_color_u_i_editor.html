Detailed Description

Provides an enhanced T:System.Drawing.Design.UITypeEditor for visually picking a color.

Includes several ways to enter a color, including transparency. Pressing Escape returns without changing the color (e.g. Cancel). Pressing Enter or clicking outside of the editor will commit the color (e.g. Ok). Editor popup (and all child controls) resizes to fit the column width.
Usage: 

    [Editor(typeof(ColorUIEditor), typeof(UITypeEditor))]
    public Color MyColor { get; set; }
