Robust data conversion. Never throws an exception. Returns the type's default value instead. Null if they are nullable types.

Template Parameters
TType of object to convert to
Parameters
valueObject to convert
Returns
Converted result

Handles everything System.Convert.ChangeType() does plus:
• Anything ==> trimmed string
• string/number (True/False, t/f, 1/0, 1.0/0.0, Yes/No (in many languages)) ==> Boolean
• Office Automation Double/float/decimal <==> DateTime/DateTimeOffset
• int(seconds since 1/1/1970) <==> DateTime/DateTimeOffset/Timespan
• long(ticks) <==> DateTime/DateTimeOffset/Timespan
• Numeric string (yyyyMMddhhmmssfff) ==> DateTime/DateTimeOffset
• System.Type <==> string
• System.Guid <==> string
• System.Version <==> string
• [Flags] System.Enum <==> string/integer

Convert CSV formatted string into an enumerable array of data models.

Template Parameters
TType of class objects to write into.
Parameters
csvStringA multi-line CSV string. Must have at least 2 rows and 2 columns.
Returns
An enumerable list of class objects.

• The column headings in the CSV stream must be of the same names as the model property name. Mismatched names are ignored. • The CSV column order is not important. • Does not perform any language translation. • CSV values that cannot be converted to the model column type will default to the default value for that data type.. • https://github.com/ChuckHill2/CsvExcelExportImport for a mor comprehensive conversion.

TextWriter  textwriter 
)
static

Serializes a single enumerable object into a single CSV document within the specified Stream. There are NO limits to the number of records written to the stream. There is no caching or buffering. Individual values are formatted and written to the stream immediately.

Parameters
itemsEnumerable list of items to write.
textwriterOpen stream to write to. Note: stream is not closed and stream pointer is not reset to beginning in order to potentially perform further processing.

• Writes columns in the same order as declared in the data model. • Does not perform any custom formatting or language translation. • https://github.com/ChuckHill2/CsvExcelExportImport for a mor comprehensive conversion.
