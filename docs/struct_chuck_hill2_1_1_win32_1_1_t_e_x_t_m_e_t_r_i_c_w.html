Detailed Description

Values are in Logical Units. For the default mapping mode, MM_TEXT, 1 logical unit is 1 pixel. For the desktop, the mapping mode is MM_TEXT. See Win32 'int GetMapMode(HDC hdc)'. For all intents and purposes, the values are in pixels.

See https://docs.microsoft.com/en-us/windows/win32/api/wingdi/ns-wingdi-textmetricw
