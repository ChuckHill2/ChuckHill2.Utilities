Detailed Description

Split screen color preview panel. The left side is the nearest known color and the right side is the current color. Clicking on the panel will set the current color to the nearest known color. It is up to the caller to subscribe to the click event to handle the new color. A tooltip may be shown displaying the name of the two colors.
