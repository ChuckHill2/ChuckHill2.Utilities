◆ SetDomainCulture()

static void ChuckHill2.Internationalization.SetDomainCulture ( string  cultureName = null)
static

Set this thread and entire AppDomain to the specified culture. Must be called at the beginning of every AppDomain startup. Use Internationalization.Dispose() to reset back to computer system culture.

Parameters
cultureNameITEF culture name or ("","iv","iv-IV") for the invariant culture. If null, retrieves the culturename from the environment variable "[productname]Culture". If still null, this function does nothing and the app defaults to the current culture of the computer.
