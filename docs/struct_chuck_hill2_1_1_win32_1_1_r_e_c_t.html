Detailed Description

Used everywhere when using Win32 RECT's via pInvoke. Also use this instead of System.Drawing.Rectangle when XML serializing because Rectangle does not serialize well. Includes implicit conversion between this RECT structure and System.Drawing.Rectangle and System.Drawing.RectangleF.
