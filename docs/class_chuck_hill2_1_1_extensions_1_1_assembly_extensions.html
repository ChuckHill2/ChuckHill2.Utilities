Gets the build/link timestamp from the specified assembly file header.

Parameters
asmAssembly to retrieve build date from
Returns
The local DateTime that the specified assembly was built.

WARNING: When compiled in a .netcore application/library, the PE timestamp is NOT set with the the application link time. It contains some other non- timestamp (hash?) value. To force the .netcore linker to embed the true timestamp as previously, add the csproj property "<Deterministic>False</Deterministic>".
