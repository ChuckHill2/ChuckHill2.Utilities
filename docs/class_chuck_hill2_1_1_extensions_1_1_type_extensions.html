Handy way to get an manifest resource stream from an assembly that 'type' resides in. This will not access Project or Form resources (e.g. *.resources).

Parameters
tType whose assembly contains the manifest resources to search.
nameThe unique trailing part of resource name to search. Generally the filename.ext part.
Returns
Found resource stream or null if not found. It's up to the caller to load it into the appropriate object. Generally Image.FromStream(s)

See ImageAttribute regarding access of any image resource from anywhere.
