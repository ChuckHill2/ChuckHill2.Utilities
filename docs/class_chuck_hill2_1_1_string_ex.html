static Func<T, string> ChuckHill2.StringEx.GetFormatter< T > ( string  formatString)
static

Create a formatter func from an interpolated format string where all the variables come from a single class. Variables are case-insensitive, whitespace trimmed, and handles unknown/missing properties. Literal "\r", "\n", "\t", and "\\" strings are converted to their character equivalants.

Internally, C# compiler converts interpolation string into a string.Format call. We do the same here at runtime.
