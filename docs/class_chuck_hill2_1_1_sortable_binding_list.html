Detailed Description

Use this instead of List<T> and BindingList<T>, as a DataSource for DataGridView object. It provides built-in support for sorting columns in a DataGridView. If there are any non-default column or row attributes (eg. font, color, etc) they will need to be reset within the DataGridView.Sorted event.
