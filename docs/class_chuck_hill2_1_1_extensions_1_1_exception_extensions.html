static Exception ChuckHill2.Extensions.ExceptionExtensions.WithStackTrace ( this Exception  ex)
static

Add a stack trace to this exception if it does not already have one. An uninitalized stack trace occurs when the caller instantiates a new Exception(). Normally, when the exception gets thrown, the stack trace is automatically added. This extension method is only needed when the newly created exception is NOT going to be thrown. This extension method will NOT overwrite a pre-existing stack trace.
