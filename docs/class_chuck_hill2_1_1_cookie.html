Detailed Description

The single public method for retrieving a browser cookie for a specific domain/host. This directly reads the browser's cookie file. The browser does not need to be running.

static string ChuckHill2.Cookie.Get ( string  domain)
static

Get browser cookie string for a specific domain from the first browser it is found in, from a list of browsers. These include Google Chrome, Microsoft Edge, Mozilla Firefox, plus many other chromium-based browsers as of 2021. Apple Safari not supported.

Parameters
domainDomain/Host name to retrieve. ex: www.google.com or .google.com or google.com (leading '.' assumes all subdomains)
Returns
Browser cookie string with domain key-value pairs delimited by semi-colons or empty if not found.

Get browser cookie string for a specific domain from the first browser it is found in, from a list of browsers. These include Google Chrome, Microsoft Edge, Mozilla Firefox, plus many other chromium-based browsers as of 2021. Apple Safari not supported.

Parameters
domainDomain/Host name to retrieve. ex: www.google.com or .google.com or google.com (leading '.' assumes all subdomains)
preferredBrowserOptional preferred browser to search first. Upon return, contains the name of the browser this cookie was retrived from. If the returned cookie is empty, this value is undefined.
Possible values are:
• Firefox or Mozilla • Chrome or Google • Edge or Microsoft • Opera • Chromium • Brave • Epic • Amigo • Vivaldi • Dragon • Torch • Comodo • Slimjet • Orbitum • Kometa • 360Browser • Maxthon3 • K-Melon • Sputnik • Nichrome • CocCoc • Uran or uCozMedia • Chromodo • Yandex or YandexBrowser
Returns
Browser cookie string with domain key-value pairs delimited by semi-colons or empty if not found.
