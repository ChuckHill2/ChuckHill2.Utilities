◆ Heartbeat

HeartbeatHandler ChuckHill2.Forms.InactivityTimer.Heartbeat
static

Handler that is called periodically (e.g. every 10% of the timeout duration). Check the handler event args for the inactivity status. Use Suspend/Resume to temporarily disable/enable the inactivity timer. This handler is called on a separate thread, so for Forms UI, Control.Invoke() must be used in order to operate upon the form.
