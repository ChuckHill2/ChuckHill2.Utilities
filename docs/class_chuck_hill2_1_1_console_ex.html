static void ChuckHill2.ConsoleEx.CopyToFile ( string  filename)
static

Split console output stream to a file. Only captures output written to console from within C# environment (e.g. Console.Writeline()). Output written via Win32 API are NOT captured! Writes are NOT blocking and are written lazily.

Parameters
filenameFilename to copy console output to. If null or empty, the existing output to file is closed. Output continues ONLY to the console. May throw an exception if unable to open the file.

Sets console window attributes more conducive to an output status window.

Parameters
titleConsole window title
backgroundColorColor of window background
textColorColor of console text
Returns
false if there is NO associated console window

This consists of not only setting the caller-specified title and window colors, but also (1)setting the font to a smaller size (still very readable). (2)Enabling select text and copy to clipboard. (3)Setting the buffer to a very large size (256x9999) to support a lot of scrolling status text. (4)Setting the window size to a slightly larger size (reasonable with the smaller font) to make the scrolling status text visible longer before it scrolls off the window.
