Detailed Description

Display a simple message dialog while executing a time intensive operation.

The dialog is blocking (e.g. user cannot perform any actions on the parent form), however the parent form itself is not blocked from any programatic actions. E.g. the message pump is still running. It is possible to dynamically update the parent form while running, but care must be taken to handle cross-thread UI actions.
