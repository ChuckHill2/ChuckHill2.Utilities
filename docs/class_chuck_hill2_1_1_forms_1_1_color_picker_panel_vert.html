Detailed Description

A Color picker/chooser panel that allows the user to visually select a color by modifying various color properties and see the results live in the preview. This resizable panel is oriented vertically.

Note: When a color swatch has focus, pressing enter will popup the standard color dialog to change the color of the selected item.
