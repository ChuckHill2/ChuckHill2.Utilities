Detailed Description

Base class to a custom-crafted COM interface class. This enables the developer to put a friendlier face on a instantiated COM object. Also for some COM objects, the automatically generated interop is incomplete, so this would be the only means. Does not support COM interfaces with '[MethodImpl(MethodImplOptions.InternalCall)]' attribute.
