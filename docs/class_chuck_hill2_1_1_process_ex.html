Parameters

timeoutSecondsDuration the executable is allowed to run before it is killed. If zero, do not wait for completion.

Run an executable and DO NOT wait for it to complete. Throws no exceptions. Any errors are written to 'sbStderr'.

Parameters
exeFull path to executable or batch file
commandLineOptional command line arguments. Note: args that contain whitespace MUST be quoted!
cwdOptional startup working directory. If null, defaults to current working directory.
sbStdoutAll console output is captured here. If null, no output is captured.
sbStderrAll console error output is captured here. If null, errors are redirected to 'sbStdout'.
Returns
Running process object. Null if Exec() itself failed (see sbStderr for message). Use p.WaitForExit(ms) to wait for completion. Then use p.ExitCode for the process exit status.. Be sure to Dispose the process object when no longer needed..
